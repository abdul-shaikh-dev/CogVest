Final Option B direction

Calm chart context for V1 Monthly Progress.

This is the agreed chart direction for issue #122: Option C's useful information density, reduced into a calmer V1 screen. The charts explain portfolio progress without becoming a spreadsheet or trading terminal.

Chart-local timeframes Each chart owns its `3M`, `6M`, `1Y`, and `All` controls so the user does not scroll away to change range.
Signed value gap `+15.8%` means portfolio value is ahead of invested value. Negative values should use red.
Value move `Value move` is the inferred monthly change after subtracting monthly investment. It avoids overclaiming market return.
Asset movement The asset headline uses relative change, while rows keep exact INR movement and allocation share shift.
Readable axes Both charts show y-axis value labels and month names. This should carry into the app implementation.
Stored snapshots only All values must derive from monthly snapshots. Preview numbers are illustrative design data only.
