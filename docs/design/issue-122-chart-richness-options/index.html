Issue #122 design choice

Monthly Progress charts that explain, not decorate.

Three V1-safe directions for making Progress charts richer. Each one keeps stored monthly snapshots as the source of truth, excludes cash from the asset trend chart, and avoids spreadsheet-style density.

Pick the data language first. Implementation should follow one of these patterns before code starts: value-gap ledger, driver narrative, or compact analyst.
